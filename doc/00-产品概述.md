# 00 · 产品概述

## 1. 一句话定位
入境游定制协作工作台——给「一手地接社 / 境外旅行社 / 省地接社」三方装上**结构化协作引擎**，让微信里的散乱沟通变成有版本、有归属、可回溯的行程+报价协同。

## 2. 目标用户（三方角色）
| 角色 | 痛点 | 系统给什么 |
|---|---|---|
| 一手 PandaKing（地接社总部） | 多方报价对不齐、利润算不清 | 成本①②分离、确认/采纳、游客报价生成 |
| 境外旅行社 | 不会规划行程、来回扯皮 | 发起结构化行程草案、加价出游客报价 |
| 省地接社 | 成本询价靠喊价 | 成本询价 H5 闭环填写成本价 |

## 3. 核心价值
- **微信不替代，只增强**：每个路线/修改/报价生成带版本号的 H5 链接，粘贴微信即结构化。
- **双向协作**：旅行社可发起草案 → 一手确认或回传修改反馈 → 循环收敛。
- **利润透明**：成本①②强制分离，模式 A+B 并存，游客报价一键生成。
- **知识沉淀 + 获客**：知识库沉淀 SOP；案例展示脱敏后对外获客（MVP 一期含）。

## 4. 技术栈（一句话）
Vue3+Vite 前端 / NestJS+PostgreSQL 后端 / CloudBase 云托管+EdgeOne CDN / 微信 OAuth2 鉴权 / EN·ZH·TH·RU 四语。
