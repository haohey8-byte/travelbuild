# 页面清单：路线管理

> 对应 `doc/02-模块拆分/路线管理.md`。最复杂模块，单列详述。

## 1. 看板视图（route-kanban）
| 区域 | 元素 | 交互 |
|---|---|---|
| 顶部统计条 | 各状态计数（待一手确认/待旅行社修订/待报价/待反馈/待确认/已确认/已流失） | 读 |
| 角色切换器 | 一手 / 旅行社 / 省地接社 | 切换视角（过滤可见字段） |
| 筛选下拉 | 按状态筛选 | 联动卡片 |
| 卡片 | 客户名(中英文)/目的地/版本/状态标签/模式标签/角标(待一手确认·待修订·有草案)/最后操作时间 | 点击开详情 |
| 新建按钮 | [+ 新建路线] | 开新建抽屉 |

## 2. 列表视图（route-list）
| 区域 | 元素 | 交互 |
|---|---|---|
| 搜索框 | 客户名/目的地/旅行社 | 实时过滤 |
| 状态筛选 chips | 全部/待一手确认/待旅行社修订/待报价/待反馈/待确认/已确认/已流失（带计数） | 切换 |
| 表格列 | 客户名(中英文)/国籍/目的地/旅行社/人数/出行日期/版本/状态/模式/待办/最后操作 | 行点击开详情 |

## 3. 路线详情 —— Tab1 行程报价（itinerary-quote）
| 区域 | 元素 | 交互 |
|---|---|---|
| 行程天编辑器 | 每天：城市/景点/住宿/餐饮/交通（可增删天） | 编辑、加天、删天 |
| 5 级报价表 | 车型/住宿/门票/餐饮/其他 × 成本①/成本②/旅行社加价/游客价 | 输入、自动算合计 |
| 成本分离 | 成本①(省地接社)、成本②(一手利润) 独立字段 | 仅一手可见 |
| 行程↔报价联动 | 改行程自动提示报价可能变化 | 智能检测变更备注 |
| 按钮栏 | [发起省地接社协作][保存草稿][保存并通知] | 选择省地接社后一键生成统一 H5 链接（可编辑行程+填成本①） |

## 4. 路线详情 —— Tab2 客户信息（customer-info）
| 区域 | 元素 | 交互 |
|---|---|---|
| 客户档案 | 姓名(中英文)/国籍/联系方式/人数/出行时间/旅行社/创建时间 | 读（一手+对应旅行社） |
| 行程草案(旅行社) | 结构化草案：按天城市/景点/住宿偏好/餐饮偏好 + 预算区间 | 旅行社可编辑重发 |
| 版本历史 | v1, v2… 列表 + 差异标注 | 查看 |

## 5. 路线详情 —— Tab3 沟通时间线（comm-timeline）
| 区域 | 元素 | 交互 |
|---|---|---|
| 时间线 | 沟通记录流（谁/何时/关于哪版/内容） | 读 |
| @引用 | 引用具体版本/报价项 | 新增时可选引用 |
| 输入框 | 文本 + 关联版本/条目 | 发送（写入时间线） |
