# 页面清单：知识库 / 账号管理 / 案例展示

## 知识库（kb）
| 页面 | 元素 | 交互 |
|---|---|---|
| 列表/搜索 | 卡片(标题/分类/标签) + 搜索 + 标签筛选 | 新建、打开 |
| 详情/编辑 | 表单(标题/分类/标签/正文富文本) + 关联路线选择器 | 保存、发布、下线 |

## 账号管理（account）
| 页面 | 元素 | 交互 |
|---|---|---|
| 成员列表 | 表格(姓名/角色/状态/最后活跃) + [邀请] | 邀请、改角色、停用 |
| 角色权限 | 字段级矩阵(角色×字段 可见性) | 只读 |
| 邀请弹窗 | 邮箱/角色/有效期 | 生成邀请链接 |

## 案例展示（cases）
| 页面 | 元素 | 交互 |
|---|---|---|
| 案例库 | 卡片(目的地/天数/主题/价格区间) + [从路线派生] | 发布、下线、分享 |
| 编辑/合规校验 | 脱敏字段表单 + 合规开关 + 二次确认弹窗 | 保存、发布 |
| 公开 H5 | 响应式案例页 + 分享(链接/二维码) | 浏览、发起咨询 |

> 详细业务规则见 `doc/02-模块拆分/` 对应模块文档。
