# 接口契约：权限矩阵（字段级）

## 角色 × 字段可见性
| 字段 | 一手 PandaKing | 境外旅行社 | 省地接社 |
|---|---|---|---|
| 客户档案 | ✓ | ✓(自身) | ✗ |
| 行程草案(旅行社) | ✓ | ✓ | ✗ |
| 成本①(省地接社成本) | ✓ | ✗ | ✓(自身) |
| 成本②(一手利润) | ✓ | ✗ | ✗ |
| 旅行社加价 | ✓ | ✓ | ✗ |
| 游客报价 | ✓ | ✓ | ✗ |
| 自身被询价成本价 | ✓ | ✗ | ✓(自身) |
| 他省地接社成本价 | ✓ | ✗ | ✗(隔离) |
| 知识库 | ✓(读写) | ✓(读) | ✗ |

## 实现要点
- 字段可见性在 **service 层** 用 `role` 投影，不靠前端隐藏（前端隐藏仅为 UX）。
- 省地接社隔离：查询强制 `WHERE agency_id = :currentAgencyId`。
- 所有写操作前校验 `canEdit(route, role)`。

## 端点
| Method | Path | 说明 |
|---|---|---|
| GET | `/api/permissions/matrix` | 返回当前角色可见字段集（前端渲染用） |
| GET | `/api/me` | 当前用户角色与权限摘要 |
