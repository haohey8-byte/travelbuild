# 接口契约：报价引擎

## 结构（5 级 × 成本分离）
```
Quote {
  levels: [
    { type: 'vehicle'|'hotel'|'ticket'|'meal'|'other',
      cost1: number,   // 省地接社成本（成本①）
      cost2: number,   // 一手利润（成本②）
      markup: number,  // 旅行社加价
      guestPrice: number } // = cost1 + cost2 + markup
  ],
  totalCost1, totalCost2, totalMarkup, totalGuestPrice
}
```
- **成本①、成本②强制分离字段**，DB 不合并。
- 模式 A：分别录 cost1/cost2；模式 B：录含利润 guestPrice，系统反解 cost2 = guestPrice - cost1 - markup（需用户选模式）。
- `guestPrice` 自动算，前端只读展示。

## 端点
| Method | Path | 说明 | 权限 |
|---|---|---|---|
| POST | `/api/routes/:id/quote/calculate` | 计算合计（不落库） | 一手/旅行社 |
| PUT | `/api/routes/:id/versions/:vid/quote` | 保存报价 | 一手/旅行社 |
| POST | `/api/routes/:id/cost-inquiry` | 生成成本询价 H5 链接 | 一手 |
| POST | `/api/h5/cost-inquiry/:token` | 省地接社填成本价①② | 持 token 省地接社 |
| POST | `/api/routes/:id/cost-adopt` | 一手采纳回填 | 一手 |

## 校验
- cost1/cost2 ≥ 0；guestPrice = cost1 + cost2 + markup（精度 2 位）。
- 省地接社仅能填自身被询价路线的 cost1（隔离）。
