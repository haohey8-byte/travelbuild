## 2. 模块地图（菜单 → 页面 → 核心按钮）
```
路线管理
├─ 看板视图      [新建路线][筛选状态][角色切换]
├─ 列表视图      [搜索][按状态筛选][打开详情]
└─ 路线详情
   ├─ Tab1 行程报价  [编辑行程][5级报价][发送成本询价][保存并通知]
   ├─ Tab2 客户信息  [客户档案][行程草案(旅行社)][版本历史]
   └─ Tab3 沟通时间线 [沟通记录][@引用]
知识库
├─ 列表/搜索     [新建条目][标签筛选]
└─ 详情          [编辑][关联路线]
账号管理
├─ 成员列表      [邀请][改角色][停用]
└─ 角色权限      [查看字段级矩阵]
案例展示
├─ 案例库        [新建案例(从路线派生)][发布/下线]
└─ 公开 H5       [分享链接/二维码]
```

## 3. 角色与权限概要
- **一手 PandaKing**：全量可见（含成本①②），确认/采纳/发布。
- **境外旅行社**：发起草案、加价、填客户；**不可见成本①②**。
- **省地接社**：仅见自己被询价的成本价字段；隔离其他省地接社。
- 详细矩阵见 `doc/04-接口契约/权限矩阵.md` 与 PRD 权限章节。

## 4. 不在 MVP 范围
- 支付/自助下单、富文本 CMS、AI 自动行程生成（远期）。
- 多组织租户隔离（当前为单 PandaKing 主体）。
